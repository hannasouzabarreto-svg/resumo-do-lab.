# resumo-do-lab.
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO.
Durante o Bootcamp Microsoft Azure, adquiri conhecimentos fundamentais sobre computação em nuvem e os principais serviços da plataforma Azure. Aprendi conceitos como modelos de serviço (IaaS, PaaS e SaaS), gerenciamento de recursos em nuvem, segurança, armazenamento, redes virtuais e monitoramento de serviços. Também desenvolvi uma visão prática sobre a criação e administração de soluções escaláveis, seguras e eficientes utilizando as ferramentas da Microsoft Azure.
